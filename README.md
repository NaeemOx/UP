# 🏭 Manufacturing Management System

A comprehensive web-based production order tracking and manufacturing process management system.

## ✨ Features

- 📊 Real-time production dashboard
- 📦 Production order tracking
- 🔧 BOM management
- 💰 Cost calculation
- 📱 Responsive design

## 🚀 Quick Start

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm run dev
```
